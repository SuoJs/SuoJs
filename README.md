# SuoJS

#### 介绍
SuoJS是3D可视化开发平台，是一站式数字孪生综合服务平台；提供三维场景搭建工具平台。 平台致力于帮助开发者低门槛、高效率开发数字孪生可视化应用。

#### 系统现有功能
1. 模型拖拽添加到场景
2. 在场景中缩放、旋转、移动模型
3. 动态添加灯光
4. 模型结构树展示
5. 镜面、水面效果
6. 场景后期处理（泛光、颜色饱和度）
7. 雨、雪效果
8. 故障火动画
